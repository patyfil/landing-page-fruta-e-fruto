## 01 - Instalando o Bootstrap e Implementando a Navbar &#x1F516;

* A ler o código do Bootstrap, interpretá-lo e fazer as modificações que precisamos.

## 02 - Adicionando o Carrossel e Resolvendo Bugs &#x1F516;
